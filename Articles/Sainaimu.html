Sainaimu: Autonomous SSH Auto-Blocker

2022-06-17 09:21:54

I made Sainaimu, an SSH blocker for Linux servers a few weeks ago. It works with ufw, and is purely configurable using a JSON configuration file.

Find it on my GitHub.

I feel like it could be used by any computer professional. I'm not sure if that's factual. In the future, I'll continue to make it more user friendly, as the concept is simple, with or without the use of the configuration file.
